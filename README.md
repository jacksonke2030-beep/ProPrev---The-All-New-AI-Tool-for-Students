# ProPrev

**Helping students stay organized and prevent procrastination with AI.**

**🔗 Live Demo:** https://pro-prev-assistant--jacksonke2030.replit.app/

## What It Does

ProPrev is an AI-powered assistant designed to help students complete assignments more efficiently and avoid procrastination. It provides guidance, breaks down tasks into manageable steps, answers questions, and supports productive study habits. By making schoolwork easier to manage, ProPrev helps students stay on track and meet deadlines with less stress.

## Stack

* Frontend: HTML, CSS, JavaScript
* Backend: Python
* AI: OpenAI API
* Hosting: Replit
